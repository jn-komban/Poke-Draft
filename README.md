🎮 PokéDraft

A multi-region Pokémon draft simulator where you build teams, battle gyms, unlock Elite Four, and carry Pokémon across generations with strategic gameplay and realistic win probabilities.


🚀 Features

* Draft Pokémon from Kanto, Johto, Hoenn, and Sinnoh
* Region-based progression system
* Carry-over Pokémon between regions
* Gym battles with realistic win percentages
* Elite Four and Champion battles for each region



🛠️ Tech Stack

* React + TypeScript
* Vite
* Modern UI components


📦 Installation & Setup

1. Clone the repository:

git clone https://github.com/jn-komban/Poke-Draft.git
cd Poke-Draft

2. Install dependencies:

npm install

3. Run the development server:

npm run dev

4. Open in browser:

http://localhost:5173


🏗️ Build for Production

npm run build


📁 Project Structure

* `src/` → main source code
* `components/` → UI components
* `routes/` → app routing
* `data/` → Pokémon data
